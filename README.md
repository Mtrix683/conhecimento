# Projeto Avaliação Técnica - ASP.NET MVC

Este projeto foi desenvolvido como parte de um teste técnico utilizando **ASP.NET MVC**.

## Tecnologias utilizadas

- ASP.NET MVC
- C#
- Razor
- Bootstrap
- HTML / CSS
- JavaScript

## Funcionalidades

- Cadastro de produtos
- Seleção de categoria
- Listagem de produtos

## Estrutura do projeto

Controllers → lógica da aplicação  
Models → entidades do sistema  
Views → interface com Razor  
wwwroot → arquivos estáticos (css, js, bootstrap)
